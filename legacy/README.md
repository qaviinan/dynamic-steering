# Legacy Files

This folder contains older scripts kept for reference after the repository
pivoted away from the earlier prototype and abliterated-model extraction path.

The current recommended workflow is:

1. `guide/how-to-abliterate.md`
2. `run_refusal_extraction.py`
3. `data/steering_vectors.npz`
4. `train_probe.py`

Files in this folder are preserved because they capture earlier experiments,
prototype infrastructure, or alternate extraction ideas that are no longer the
main path.

Legacy tests that still exercise those older flows remain under `tests/` and
are marked in-file.
